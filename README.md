# 정왕동 칵테일 🍹

칵테일 메뉴 제공

___
## 제작 의도
### 테스트
1. Room 라이브러리 이용한 데이터 저장 및 불러오기(+ 코루틴)
2. Room 라이브러리, LiveData, RecyclerView 연계해 사용
3. Parcelable 객체 이용한 정보 전달
4. SHA-256이용한 암호 저장
